Following the tutorial step by step to run the application in the right way:

1. Open folder local_api and run the file app.py in order to start a local server.

2. To run app.py, open 	CMD right in this folder and execute this command.

Copy this command: python app.py

3. Next, open Simple Chatbot folder and open simple_chatbot.exe file to run the demo application.

4. Just try all the features of demo app. ^^
